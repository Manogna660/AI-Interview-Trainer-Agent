# 🤖 InterviewGenius AI

## Agentic AI Interview Coach using IBM Granite

### Project Overview

InterviewGenius AI is an Agentic AI powered interview preparation platform built using IBM Granite Foundation Models and IBM watsonx.ai.

The system consists of multiple AI agents that work together to provide a personalized interview experience.

---

## AI Agents

1. Resume Analysis Agent

2. Skill Extraction Agent

3. Interview Generation Agent

4. Evaluation Agent

5. Recommendation Agent

---
